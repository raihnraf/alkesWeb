## Spacing Scale

Declared values (multiples of 4 only, from existing `@theme` tokens in `src/css/main.css`):

| Token | Value | Usage |
|-------|-------|-------|
| stack-sm | 8px | Icon-to-text gaps, inline padding, spec table cell padding |
| stack-md | 16px | Default element spacing, card internal padding, form field gaps |
| gutter | 24px | Grid column gaps, horizontal spacing between related elements |
| stack-lg | 32px | Section-to-section vertical spacing within product page content |
| section-padding | 80px | Major section top/bottom padding (hero, specs, applications, CTA, related) |
| margin-mobile | 16px | Horizontal page margin on mobile (<768px) |
| margin-desktop | 48px | Horizontal page margin on desktop (>=768px) |
| container-max | 1280px | Max content width for all product page sections |

Exceptions:
- Touch targets: minimum 44px height for all interactive elements (WhatsApp button, email CTA, related product links, hamburger menu). Already enforced in Phase 2 shared layout.
- Hero image container: min-height 320px on mobile, 480px on desktop to ensure product image prominence per PROD-02.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#f9f9ff` (surface) | Page background, content area surfaces |
| Secondary (30%) | `#ffffff` (surface-container-lowest) | Cards, spec table rows, CTA banner background |
| Accent (10%) | `#00A3AD` (medical-teal) | WhatsApp button, section divider bars, icon accents, active states, hover states on links |
| Deep Blue | `#003366` (deep-blue) | Headings, footer background, email CTA button, nav bar |
| Destructive | `#ba1a1a` (error) | Not used in this phase — no destructive actions on product pages |
| Alert Gold | `#FFB800` (alert-gold) | Purchase/rental banner icon badge (availability indicator) |

**60/30/10 split applied per product page:**
- 60% surface (`#f9f9ff`) — page background between sections
- 30% white cards (`#ffffff`) — spec table, clinical application cards, related product cards
- 10% medical-teal (`#00A3AD`) — WhatsApp CTA, section accent bars, icon highlights, link hovers

**Accent (`#00A3AD`) reserved for:**
1. WhatsApp inquiry button background
2. Section divider accent bars (32px wide, 4px tall, above section headings)
3. Specification table row hover highlight
4. Related product card hover border color
5. Icon color in clinical application cards
6. Active navigation state (already defined in Phase 2)

## Layout Contract

Each of the 4 product pages follows this vertical section order:

### Section 1: Product Hero (PROD-02)
- Full-width section below fixed header, background: `bg-deep-blue`
- Grid layout: 2-column on desktop (image left 50%, text right 50%), single column on mobile (image top, text bottom)
- Product image: `object-contain` on white/light background area, max-width 520px, with subtle shadow
- Product name: `headline-xl` (desktop) / `headline-lg-mobile` (mobile), white text
- Product tagline/description: `body-lg`, `text-surface-variant`, max-width 2xl
- Breadcrumb: `label-md`, `text-surface-variant/60`, above product name: "Produk > [Product Name]"
- Min-height: 480px desktop, 320px mobile

### Section 2: Specifications Table (PROD-03)
- Background: `bg-surface-container-low` (`#f0f3ff`)
- Container: `max-w-[--spacing-container-max] mx-auto px-[--spacing-margin-mobile] md:px-[--spacing-margin-desktop]`
- Section heading: `headline-lg`, `text-deep-blue`, with medical-teal accent bar (32px × 4px) above
- Table: 2-column layout (spec name | spec value), full-width within container
  - Header row: `label-md`, `text-deep-blue`, `bg-white`, `border-b-2 border-medical-teal`
  - Data rows: `body-md`, alternating `bg-white` / `bg-surface-container-lowest`, `border-b border-border-light`
  - Row hover: `bg-medical-teal/5`
  - Cell padding: `py-3 px-4`
- Mobile: table collapses to stacked key-value pairs (label on top, value below)

### Section 3: Clinical Applications (PROD-04)
- Background: `bg-surface` (`#f9f9ff`)
- Section heading: `headline-lg`, `text-deep-blue`, with medical-teal accent bar above
- Card grid: 1-column mobile, 2-column tablet, 3-column desktop
- Each card:
  - Icon: Material Symbols Outlined, 36px, `text-medical-teal`
  - Title: `headline-md`, `text-deep-blue`
  - Description: `body-md`, `text-on-surface-variant`
  - Card style: `bg-white rounded-xl border border-border-light p-6`, hover shadow `shadow-[0px_4px_20px_rgba(0,51,102,0.05)]`

### Section 4: Purchase/Rental Banner (PROD-05)
- Background: `bg-deep-blue` (full-width banner)
- Container: same max-width + padding as other sections
- Layout: flex row, items center, gap gutter
- Icon: alert-gold circle badge (`w-12 h-12 rounded-full bg-alert-gold/20 flex items-center justify-center`)
- Heading: `headline-md`, `text-white` — "Tersedia untuk Pembelian atau Penyewaan"
- Body: `body-md`, `text-surface-variant/80` — brief description
- CTA buttons inline: WhatsApp + Email (see CTA section below)
- Shown only on applicable products (Blue Diode Laser, Ultrasonic Surgery, Laser Holmium). Hidden on Consumable Urologi page.

### Section 5: Inquiry CTAs (PROD-06, PROD-08)
Two buttons displayed side-by-side (stacked on mobile):

**WhatsApp Button (primary):**
- Style: `bg-medical-teal text-white px-8 py-3 rounded-full font-label-md text-label-md`
- Hover: `bg-white text-medical-teal`
- Icon: Material Symbols `chat` (20px), left of text
- Label: "Tanya via WhatsApp"
- Link: `https://wa.me/6281112129779?text=Halo%20APBA%20Group%2C%20saya%20tertarik%20dengan%20[Product%20Name].%20Mohon%20informasi%20lebih%20lanjut.`
- Target: `_blank`, `rel="noopener noreferrer"`
- Min-height: 44px

**Email Button (secondary):**
- Style: `border-2 border-deep-blue text-deep-blue px-8 py-3 rounded-full font-label-md text-label-md`
- Hover: `bg-deep-blue text-white`
- Icon: Material Symbols `mail` (20px), left of text
- Label: "Kirim Email"
- Link: `mailto:info@apbagroup.com?subject=Inquiry%20[Product%20Name]&body=Halo%20APBA%20Group%2C%0A%0ASaya%20tertarik%20dengan%20[Product%20Name].%20Mohon%20informasi%20lebih%20lanjut.%0A%0ANama%3A%20%0AInstitusi%3A%20%0A%0ATerima%20kasih.`

### Section 6: Related Products (PROD-07)
- Background: `bg-surface-container-low` (`#f0f3ff`)
- Section heading: `headline-lg`, `text-deep-blue`, with medical-teal accent bar above
- Card grid: 1-column mobile, 2-column tablet, 3-column desktop (shows 3 other products)
- Each card:
  - Thumbnail: product image, `h-40`, `bg-background-subtle`, `object-contain`, `p-4`
  - Title: `headline-md`, `text-deep-blue`
  - Link: full card clickable, `text-medical-teal` with arrow icon
  - Card style: `bg-white rounded-xl border border-border-light overflow-hidden hover:shadow-[0px_4px_20px_rgba(0,51,102,0.05)] transition-all duration-300`
  - Hover: border color changes to `border-medical-teal`

## Interaction Contract

| Interaction | Behavior |
|-------------|----------|
| WhatsApp button click | Opens `wa.me` in new tab with pre-filled message containing product name |
| Email button click | Opens mail client with pre-filled subject line containing product name and body template |
| Spec table row hover | Background changes to `bg-medical-teal/5` for visual scanning |
| Application card hover | Shadow increases to `shadow-[0px_4px_20px_rgba(0,51,102,0.05)]` |
| Related product card hover | Shadow increases + border color changes to medical-teal |
| Related product card click | Navigates to the clicked product's detail page |
| Breadcrumb "Produk" click | Navigates to home page `#products` section |
| All CTAs | Min 44px touch target, visible focus ring (`focus-visible:ring-2 focus-visible:ring-medical-teal`) |

## Accessibility Contract

| Requirement | Implementation |
|-------------|---------------|
| Skip link | Already present in Phase 2 shared header |
| Alt text on product images | Descriptive alt text per product (e.g., "Mesin Blue Diode Laser untuk prosedur urologi — tampilan depan dengan panel kontrol digital") |
| Focus visible | All interactive elements: `focus-visible:ring-2 focus-visible:ring-medical-teal focus-visible:ring-offset-2` |
| Color contrast | All text meets WCAG AA 4.5:1 minimum (deep-blue on white = 12.6:1, medical-teal on white = 3.2:1 — use medical-teal only for large text/icons or on dark backgrounds) |
| Semantic HTML | `<main>`, `<section>`, `<h1>`–`<h3>` hierarchy, `<table>` with `<thead>`/`<tbody>`/`<th>` for specs |
| Reduced motion | Already handled in Phase 1 `main.css` via `prefers-reduced-motion` media query |
