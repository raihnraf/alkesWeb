## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-05 | Tampering | Image file path traversal | mitigate | Images served from public/ at root path. Vite prevents parent directory access. Always use /images/ paths, not ../public/images/ (per RESEARCH.md Pitfall 3). |
| T-01-07 | Denial of Service | Large image files causing slow builds | accept | 4 PNG images at ~3MB total. vite-plugin-image-optimizer will compress and convert to WebP, reducing size. Acceptable for Phase 1. |
| T-01-08 | Tampering | Image optimizer processing malicious files | mitigate | vite-plugin-image-optimizer uses Sharp which validates image formats. Only processes PNG/JPEG from public/images/. No user uploads in Phase 1. |
</threat_model>
