## Color

Migrated from DESIGN.MD Tailwind v3 config to Tailwind v4 @theme directive:

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #f9f9ff (surface/background) | Page background, main surfaces |
| Secondary (30%) | #003366 (deep-blue, primary-container) | Cards, navigation, footer, sections |
| Accent (10%) | #00A3AD (medical-teal) | CTAs, links, active states, highlights, icons |
| Destructive | #ba1a1a (error) | Error states, destructive actions |
| Alert | #FFB800 (alert-gold) | Warnings, attention states |

**Accent reserved for:**
- Primary CTA buttons ("Lihat Produk Kami", "Hubungi Kami")
- Navigation active states
- Link hover states
- Icon fills
- Stat counters
- Feature section highlights
- Form focus rings

**NOT for:** Body text, borders, backgrounds (use dominant/secondary)

**Source:** DESIGN.MD lines 18-71 (existing 50+ color tokens)

## Design Token Migration Strategy

Phase 1 migrates DESIGN.MD Tailwind v3 JavaScript config to Tailwind v4 CSS @theme directive:

**Before (v3 JavaScript):**
```javascript
theme: {
  extend: {
    colors: {
      primary: "#001e40",
      "medical-teal": "#00A3AD",
      // ... 50+ color tokens
    },
    spacing: {
      "section-padding": "80px",
      // ... spacing tokens
    }
  }
}
```

**After (v4 CSS):**
```css
@import "tailwindcss";

@theme {
  --color-primary: #001e40;
  --color-medical-teal: #00A3AD;
  --spacing-section-padding: 80px;
  --font-headline-xl: "Plus Jakarta Sans", sans-serif;
  /* ... all 50+ tokens migrated */
}
```

**Migration completeness:** All 50+ color tokens, 8 spacing tokens, 8 font size/weight combinations, 4 font families
