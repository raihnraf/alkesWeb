## Spacing Scale

Declared values from `src/css/main.css` `@theme` block (lines 58-66). All multiples of 8 except 4px edge case:

| Token | Value | Usage in this phase |
|-------|-------|---------------------|
| `--spacing-stack-sm` | 8px | Nav link internal gaps, icon-to-text spacing |
| `--spacing-stack-md` | 16px | Footer column item gaps, mobile menu item padding |
| `--spacing-gutter` | 24px | Nav link horizontal spacing, footer column gaps |
| `--spacing-stack-lg` | 32px | Header internal section gaps (logo ↔ nav ↔ CTA) |
| `--spacing-margin-desktop` | 48px | Header/footer horizontal padding (desktop ≥1024px) |
| `--spacing-margin-mobile` | 16px | Header/footer horizontal padding (mobile <768px) |
| `--spacing-section-padding` | 80px | Not used in header/footer; reserved for page sections |

**Exceptions:**
- Touch target minimum: 44px × 44px for hamburger button and mobile menu items (WCAG 2.5.5), overrides `--spacing-stack-sm` for interactive element sizing
- Header height: 80px desktop (fixed), 64px mobile — derived from DESIGN.MD `h-20` class

## Active State Indicators

### Desktop Active Nav Link

| Property | Value |
|----------|-------|
| Text color | `text-medical-teal` |
| Font weight | `font-bold` (700) — overrides default 600 |
| Bottom border | `border-b-2 border-medical-teal` |
| Padding bottom | `pb-1` (to accommodate border without layout shift) |
| Detection | JavaScript: compare `window.location.pathname` with link `href`; add `.active` class |

### Mobile Active Menu Item

| Property | Value |
|----------|-------|
| Text color | `text-medical-teal` |
| Font weight | `font-bold` (700) |
| Background | `bg-medical-teal/5` (5% teal tint) |
| Left border | `border-l-4 border-medical-teal` |
| Padding left | `pl-[calc(--spacing-margin-mobile-4px)]` to accommodate border |

### Active State Detection Logic

```javascript
// Run on each page load
const currentPath = window.location.pathname;
const navLinks = document.querySelectorAll('nav a[href]');
navLinks.forEach(link => {
  const linkPath = new URL(link.href).pathname;
  if (linkPath === currentPath || (currentPath === '/' && linkPath === '/')) {
    link.classList.add('active');
  }
});
```

**Note:** For product pages (`/products/*`), the "Produk" nav link should be active when any product sub-page is current. Detection: `currentPath.startsWith('/products/')`.

**Source:** Success criterion #2 (current page visually indicated), DESIGN.MD line 198 (active state example)

## Animation Specifications

| Animation | Trigger | Properties | Duration | Easing |
|-----------|---------|------------|----------|--------|
| Mobile menu open | Hamburger click (closed → open) | `max-height: 0 → calc(100vh - 80px)`, `opacity: 0 → 1` | 300ms | `ease-in-out` |
| Mobile menu close | Hamburger click (open → closed) / Escape / link click | `max-height: calc(100vh - 80px) → 0`, `opacity: 1 → 0` | 300ms | `ease-in-out` |
| Nav link hover | Mouse enter | `color: --color-on-surface → --color-medical-teal` | 200ms | `ease` |
| CTA button hover | Mouse enter | `background: --color-deep-blue → --color-medical-teal` | 200ms | `ease` |
| Footer link hover | Mouse enter | `color: --color-surface-variant/80 → --color-white` | 200ms | `ease` |
| Hamburger icon swap | Menu toggle | Icon text: `menu ↔ close` | Instant (no transition) | — |
| Logo hover | Mouse enter | `color: --color-deep-blue → --color-medical-teal` | 200ms | `ease` |

**JavaScript for mobile menu:**
```javascript
const hamburger = document.getElementById('hamburger-btn');
const mobileMenu = document.getElementById('mobile-menu');
const menuIcon = document.getElementById('menu-icon');

hamburger.addEventListener('click', () => {
  const isOpen = mobileMenu.classList.toggle('open');
  hamburger.setAttribute('aria-expanded', isOpen);
  menuIcon.textContent = isOpen ? 'close' : 'menu';
  document.body.style.overflow = isOpen ? 'hidden' : '';
});
```

## Accessibility Requirements

| Requirement | Implementation |
|-------------|----------------|
| Semantic HTML | `<header>`, `<nav>`, `<main>`, `<footer>` elements on every page |
| ARIA labels | Hamburger: `aria-label="Buka menu navigasi"`, `aria-expanded`, `aria-controls="mobile-menu"` |
| Keyboard navigation | Tab order: Logo → Nav links → CTA → Hamburger (desktop); Logo → Hamburger → Menu items (mobile) |
| Focus visible | All interactive elements have `:focus-visible` ring: `ring-2 ring-medical-teal ring-offset-2 ring-offset-surface` |
| Skip link | "Langsung ke konten utama" skip-to-content link, visible on focus, positioned above header |
| Color contrast | All text meets WCAG AA 4.5:1 minimum — verified: `--color-on-surface` on `--color-surface` = 16.5:1, `--color-white` on `--color-deep-blue` = 12.8:1 |
| Touch targets | Minimum 44×44px for hamburger button and mobile menu items |
| Language | `lang="id"` on `<html>` element |
| Screen reader | Mobile menu heading "Menu" with `sr-only` class for context |
| Reduced motion | Respect `prefers-reduced-motion: reduce` — disable menu animation, use instant show/hide |

**Reduced motion CSS:**
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## Cross-Page Consistency Requirements

| Element | Consistency Rule |
|---------|-----------------|
| Header | Identical HTML structure on all 7 pages; only active state class changes per page |
| Footer | Identical HTML structure on all 7 pages |
| CSS | Single `main.css` import on every page — no page-specific overrides for nav/footer |

| Page content offset | `<main>` has `pt-20` (80px) on all pages to account for fixed header |
